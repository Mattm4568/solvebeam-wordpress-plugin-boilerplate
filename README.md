## 🚀 What this is

solvebeam-wordpress-plugin-boilerplate is a WordPress plugin starter package. It gives you a clean base for building a plugin with a clear file layout and common setup files.

Use it when you want to:
- start a new WordPress plugin from a structured base
- keep files organized as the plugin grows
- follow a setup that fits modern WordPress work
- save time on the first setup steps

## 💻 Before you start

You only need a few things on your Windows PC:
- a web browser
- access to your WordPress site
- enough space to save the download
- a ZIP extractor such as File Explorer or 7-Zip

This package is meant for WordPress plugin work. If you plan to use it, you should already have WordPress installed on your site or local test setup.

## 🪟 Install on Windows

After the ZIP file finishes downloading:

1. Open File Explorer
2. Go to the folder where you saved the ZIP
3. Right-click the ZIP file
4. Choose Extract All
5. Pick a folder you can find later
6. Open the extracted folder

If you use 7-Zip:
1. Right-click the ZIP file
2. Choose 7-Zip
3. Choose Extract Here or Extract to folder name
4. Open the new folder after extraction

## 🧩 Use with WordPress

This package is a plugin base, so the next step is to place it where WordPress can use it.

If you are using a local WordPress install:
1. Open the WordPress folder on your computer
2. Go to `wp-content/plugins`
3. Copy the extracted plugin folder into `plugins`

If you are using a live site:
1. Sign in to your WordPress admin area
2. Go to Plugins
3. Upload the plugin ZIP if your site allows uploads
4. Or place the files on the server through your hosting file manager

After that:
1. Open WordPress admin
2. Go to Plugins
3. Find the plugin
4. Click Activate

A typical structure for a plugin boilerplate looks like this:

## 🧷 Troubleshooting

If the download does not start:
1. Refresh the releases page
2. Try the newest release
3. Check your browser download settings
4. Try a different browser

If the ZIP will not open:
1. Download it again
2. Check that the file finished downloading
3. Use File Explorer or 7-Zip
4. Make sure the file was not renamed

If WordPress does not show the plugin:
1. Check that the folder is in the right `plugins` directory
2. Make sure the main plugin files are inside the folder
3. Refresh the Plugins page in WordPress

If activation fails:
1. Check that the plugin folder was extracted fully
2. Make sure the files did not end up in a nested folder
3. Confirm that your WordPress version is current
